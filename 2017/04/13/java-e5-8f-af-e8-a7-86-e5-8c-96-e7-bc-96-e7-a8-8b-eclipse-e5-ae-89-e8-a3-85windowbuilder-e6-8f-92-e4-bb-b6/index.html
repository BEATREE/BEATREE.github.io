JAVA eclipse
Java可视化编程-eclipse安装windowbuilder插件

图形用户界面：Graphical User Interfaces, 简称GUI。

图形用户界面：

  • 已经成为面向用户的应用程序的基本要求
  • 几乎所有的单机版程序和客户端/服务器程序（Client/Server）的客户端部分都采用图形用户界面
  • 图形用户界面的好坏，图形界面程序的编写性能，直接影响一种编程语言的实用性

使用 java 编程的程序员们应该都知道，java创建一个窗体，并不是像 VB 和 C # 以及国产的 易语言 一样，可以直接使用可视化来进行创建窗体和窗体上的组件，即使一个简单的窗体在 java 中也需要一大串代码来进行实现。 我们今天来介绍一种可以叫做 windowbuilder 的插件，它可以辅助我们来实现 java 的可视化编程。

下边我们来说一下它的安装以及使用：

  1. 登陆eclipse官方网站，在搜索栏搜索“windowbuilder”，会跳转到windowbuilder的界面。 [caption id=”attachment_1306” align=”aligncenter” width=”520”] 搜索windowbuilder[/caption] [caption id=”attachment_1307” align=”aligncenter” width=”1161”] windowbuilder搜索结果界面[/caption]

  2. 点击进入，eclipse windowbuilder 的插件界面，然后点击第二个“Downloads”按钮，会跳出另一个界面——下载界面。

  3. 选择下载哪一个是注意你的eclipse版本，例如，你使用的 eclipse 是 mars，则选择 mars 版本对应的下载。依次类推，你使用的哪个版本，则下载相对应的版本。然后点击 “link” [caption id=”attachment_1308” align=”aligncenter” width=”1175”] 下载插件[/caption]

  1. 但是点击后并不一定会出现下载的提示，也可能是这个情况：这个情况是由于你没有打开eclipse客户端，解决的方法也十分简单，打开eclipse客户端，刷新界面。这个时候的界面是一个空白的界面，请不要关闭哦！

    注意网页的网址：它是下载地址！（下载地址）

  2. 然后切换到eclipse ，点击标题栏中的帮助并再点击安装新软件，然后会跳出另一个界面即 界面二，然后在“work with：”后输入刚才复制的网址，下边会出现其中包含的文件，选中需要下载的，不知道选哪个可以全部选中，然后下载安装即可。 [caption id=”attachment_1311” align=”aligncenter” width=”416”] 界面一[/caption] [caption id=”attachment_1312” align=”aligncenter” width=”877”] 界面二[/caption]

安装完成后，新建的话，新建项目，然后找到windowbuilder，在选择 jframe，即可创建可视化编程界面。 附：下载界面
