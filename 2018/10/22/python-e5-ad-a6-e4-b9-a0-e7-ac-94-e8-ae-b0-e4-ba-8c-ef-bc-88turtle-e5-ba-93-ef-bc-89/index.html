# 板块1：turtle库的使用

##空间坐标体系

    空间坐标体系：**绝对坐标**，以及**海龟坐标**

### 绝对坐标：刚开始，海龟默认在最中心，以海龟为中心即原点，分为X轴和Y轴

在这种坐标体系中通过 goto(x, y)进行定义

### 海龟坐标、

turtle.circle(r, angle) # 表示以海龟左侧某个点为圆心，作曲线运行
turtle.fd(d) # 表示向海龟当前指向前进
turtle.bk(d) # 表示向海龟的反方向运行

## turtle的角度坐标体系

可以使用seth进行角度的改变，但是它不进行绘制，只是改变海龟的方向

海龟角度，通过使用 left 和 right 进行方向的改变 例子：

import turtle
turtle.left(45)
turtle.fd(150)
turtle.right(135)
turtle.fd(300)
turtle.left(135)
turtle.fd(150)
turtle.done()
