WordPress上传文件时自动重命名插件：File Renaming on upload

File Renaming on upload 是一个WordPress文件上传时重命名插件，也就是在你上传文件的时候，自动重命名该文件，可以添加域名、使用日期、过滤特殊符号等来命名文件。 我们都知道，如果上传的文件是用中文命名的，很可能就会没办法正常显示。使用该插件就可以解决这个问题。 在后台插件安装界面搜索 File Renaming on upload 即可在线安装，或者下载 File Renaming on upload
