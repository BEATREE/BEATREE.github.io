Proxyee-down 百度云网盘不限速的免费开源下载神器

百度网盘（原百度云）是百度推出的一项云存储服务，首次注册即有机会获得2T的空间，用户将可以轻松将自己的文件上传到网盘上，并可跨终端随时随地查看和分享。 但它对免费用户的限速实在是严苛，即便是100M的宽带，常常也会遇到仅有几十KB/秒的速度，一个大文件不知道要下载到什么时候了。 曾经也有一些破解百度网盘限速的软件和技术，比如使用 PanDownload、Aria2 多线程下载等方法，不过这些方法多少已经失效了，或者不能再进行不限速下载了。而现在，又出现了一个更强大的免登录帐号且能突破限速下载百度云网盘的利器——Proxyee-down！

Proxyee-down 简介

Proxyee-down 是一款免费且开源的 HTTP 多线程嗅探下载工具，基于 Java 开发，跨平台支持 Windows、Mac 和 Linux 系统。最重要的是，它内置了类似百度网盘下载插件功能，可以突破速度限制，免费达到多线程高速下载的效果。 这款开源工具可帮助我们构建网页代理嗅探百度下载地址，嗅探后的地址直接在网页建立多个连接加速下载。 实测该工具在百兆带宽下下载速度可以达到6~9MB每秒！！！ Github 项目地址：https://github.com/monkeyWie/proxyee-down 感谢该项目作者的贡献： monkeyWie

Proxyee-down 的安装使用方法

Proxyee-down 的工作原理与之前的下载工具有所不同，它是通过在本机建立一个 HTTP 代理服务器，然后设置浏览器 (或操作系统) 的网络连接走这个代理，从而接管到百度网盘的下载请求，并从中嗅探出真实的文件下载地址。

安装证书

由于百度云网盘现已采用 HTTPS 加密连接，第三方工具无法直接从网络流量中截获下载请求，因此首次使用 Proxyee-down 时软件会要求你安装一个数字证书，点击确定安装，让系统信任 Proxyee-down 之后才能解密 https，从而让其正常工作。否则会出现由于链接不安全拒绝链接的情况！

设置代理模式（其选项名称为“嗅探模式”）

Proxyee-down 的代理提供了两种嗅探模式：全网嗅探 和 百度云嗅探，我们一般默认使用仅嗅探百度云即可 (仅对百度云网站使用代理)，不需要设置全局代理。 以 Windows 版本为例，启动后右键点击右下角托盘图标，选择「嗅探模式」→「百度云」即可，系统的 IE 代理设置会被 Proxyee-down 临时修改，这时，IE 浏览器、Edge 以及绝大多数支持 IE 代理的浏览器都已经可以正常使用 Proxyee-down 来接管百度网盘下载了。

如何使用

当证书和代理设置都正确并启动了 Proxyee-down 之后，通过浏览器访问百度云网盘的下载链接时，就会出现一个 proxyee-down 字样了，如下图，这时就表明 Proxyee-down 已准备就绪。 直接点击网盘页面的下载按钮，Proxyee-down 就会接管过来，开始下载任务了。剩下的选项大家自己也就明白了！
